## clubs

| File | Summary |
| --- | --- |

| Directory | Summary |
| --- | --- |
| application | The application layer provides club discovery and management services: a geolocation service for fetching user coordinates to find nearby clubs, and Riverpod providers for querying clubs, calculating distances via Haversine formula, and mutating club membership/run data with cache invalidation. |
| data | The data directory contains the Supabase repository implementation for clubs data access, providing methods to fetch, list, and search clubs from the backend. |
| domain | — |
| presentation | Presentation layer for club features, providing UI screens for listing clubs, viewing club details, creating and editing clubs, and managing club runs. |
