## src

| File | Summary |
| --- | --- |

| Directory | Summary |
| --- | --- |
| common_widgets | The common_widgets directory contains reusable Flutter UI components, including a brand header widget for app branding and a user avatar widget for displaying user profile pictures. |
| core | The core directory provides foundational app infrastructure including input validators, presentation utilities for error handling (like CopyableErrorText), and a telemetry system that collects and persists sanitized events and breadcrumbs to Supabase. |
| features | The features directory contains the core functional modules for the UFF fitness app, providing authentication and onboarding, activity import and analytics, photo management, social features including clubs and user interactions, and user preferences like settings and gear management. |
| routing | The HomeShellScreen defines the main navigation shell for the app's home section, providing a bottom navigation bar with five destinations (Feed, Activity, Record, Analytics, Profile) and an app bar with sync status indicator and action buttons for notifications, import, and settings. |
